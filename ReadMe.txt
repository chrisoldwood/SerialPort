SerialPort v0.1
===============

Introduction
------------

This is a command line (aka console) based tool for redirecting output from a
command to a serial port. This facility is not built into the shell on the early
Windows 95 family.

Releases
--------

Stable releases are formally packaged and made available from my Win32 tools page:
http://www.chrisoldwood.com/win32.htm

The latest code is available from my GitHub repo:
https://github.com/chrisoldwood/SerialPort
